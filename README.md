# Valentines
A cute message to ask your love to be your valentine !! &lt;3
To add your custom messsage to it;
1) Start your VS Code or any other IDE you use.
2) Open the full folder of the github in VS Code.
3) Open index.html
4) Edit your required texts between <h2>
5) Run using Ctrl + f5 from index.html
6) Enjoy <3
   
IMPORTANT !!!! 
The file is not hosted anywhere. So you need to share the whole source code with the person with PC access to make it run properly.
